# 지원 브라우저 및 PWA 기준

## 지원 대상

- 팀 표준 브라우저 범위를 문서화하고 일관되게 적용한다.
- 호환성 이슈는 우선순위를 정의해 대응한다.

## 비지원 범위

- 레거시 브라우저는 지원 범위에서 제외한다.

## PWA 지원 전략(요약)

- 서비스 워커 역할과 책임을 명확히 한다.
- 캐싱 대상과 제외 대상을 분리한다.
- 업데이트 정책과 무효화 정책을 문서화한다.

## 푸시 알림 적용 기준(요약)

- 적용 목적과 대상 범위를 명확히 한다.
- 보안 및 인증 기준을 준수한다.
