# Polyfill 전략

## 기본 원칙

- 필요한 기능에 한해 최소 범위로 적용한다.
- 전역 적용은 반드시 근거를 문서화한다.

## 적용 기준(요약)

- 전역 적용이 필요한 경우
- 특정 기능/페이지 한정 적용이 필요한 경우
- 적용하지 않는 경우를 명확히 정의

## PWA 환경 고려

- 오프라인 환경 및 캐시 정책과 충돌하지 않도록 한다.
- 성능 저하가 없는지 검증한다.
