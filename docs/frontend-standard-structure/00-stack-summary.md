# 스택 요약

## 기본 원칙

- 앱 구조는 Next.js App Router 기준으로 설계한다.
- 모든 코드와 문서는 TypeScript 기준을 우선한다.
- 스타일은 공통 규약에 따라 우선순위를 둔다.

## 프로젝트 기준 스택(요약)

- 런타임/프레임워크: Next.js(App Router)
- 언어: TypeScript
- 스타일: Tailwind CSS 중심 (필요 시 설계 기준에 따른 예외 허용)
- 패키지 매니저: pnpm
