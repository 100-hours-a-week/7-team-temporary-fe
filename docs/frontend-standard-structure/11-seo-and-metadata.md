# SEO 및 메타데이터 전략

## 기본 방침

- 페이지 목적에 맞는 메타데이터를 제공한다.
- 중복/충돌되는 메타 정보는 제거한다.

## 적용 기준

- 라우트 단위로 메타데이터를 관리한다.
- 기본 메타 + 페이지별 메타로 구성한다.
